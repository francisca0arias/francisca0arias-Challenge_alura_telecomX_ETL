# francisca0arias-Challenge_alura_telecomX_ETL
La limpieza y tratamiento de datos garantizan precisión en el análisis, corrigiendo valores nulos o duplicados. El análisis exploratorio permite descubrir patrones y relaciones mediante estadísticas y visualizaciones, generando insights estratégicos que fortalecen decisiones empresariales.
Análisis Predictivo de Churn: Solución Integral Telecom X
🎯 Objetivo General
Desarrollar una infraestructura de datos robusta mediante un pipeline de ETL (Extract, Transform, Load) y un Análisis Exploratorio de Datos (EDA) profundo. El fin es transformar volúmenes de información en "Single Source of Truth" para identificar los drivers críticos de la evasión de clientes y sustentar decisiones estratégicas de negocio.

🛠️ Arquitectura de la Solución
1. Dominio del ETL (Calidad de Datos)
La limpieza y tratamiento son la base de la confiabilidad. Se implementó un proceso de ingeniería de datos enfocado en:

Identificación de Inconsistencias: Manejo de valores nulos y duplicados en un dataset de +7,000 registros.

Tipado Dinámico: Conversión de variables como TotalCharges para asegurar la integridad de los cálculos.

Estandarización: Normalización de categorías para eliminar datos fuera de estándar.

2. Análisis Exploratorio (EDA) y Visualización
Capacidad analítica aplicada para descubrir patrones ocultos mediante estadísticas descriptivas:

Segmentación Crítica: Hallazgo de una tasa de churn del 42.7% en contratos mensuales frente a un 2.8% en contratos bienales.

Drivers de Fricción: Identificación de problemas de retención en usuarios de fibra óptica (41.9% de churn) y métodos de pago manuales.

📈 Resultados e Insights Estratégicos
El análisis arroja tres pilares de acción inmediata:

Migración Contractual: Incentivar el paso de planes mensuales a anuales.

Optimización de CX: Programas de retención intensivos durante los primeros 18 meses de vida del cliente.

Transformación Digital: Migrar el método de pago "Electronic Check" a procesos automáticos para reducir la rotación pasiva.
